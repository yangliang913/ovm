Linux ISO Install
Linux Installation from ISO

Type: ConVirt Provisionable Template

Version: 2.0

Author: The ConVirt Team

Requirements: ConVirt v3.2+, ESXi5.0

Description: This template creates a VM with 10G virtual disk for use as the primary hard drive. The VM boots from a bootable Linux ISO which, in turn, should kick off the distribution specific installation routine and deploy the OS on the primary hard drive.

Instructions:
1. Check the default provisioning parameters and alter them to suit your environment if necessary
2. (Important) The default location for VM is determined by DATA_STORE variable. Alter DATA_STORE value in the Template Parameter to match your environment.
2. (Important) The default location for ISO is determined by ISO_STORE variable. Alter ISO_STORE value in the Template Parameter to match your environment.
3. (Important) The actual ISO name is specified in the disk property, clieck on the second disk and specify the name of the full iso file.
4. Provision the VM configuration normally using ConVirt.
